Será esta a arma secreta da Google para ter a melhor câmara do mercado no Pixel 3 XL?

Com os principais fabricantes de smartphones a apresentarem grande evoluções na fotografia, a Google não quer ficar atrás.
Será esta a arma secreta da Google para tornar o Pixel 3 XL o equipamento com melhor câmara do mercado?

Nos últimos anos tempos assistido a uma grande aposta na componente fotográfica, motivando uma verdadeira guerra entre os fabricantes para que os seus equipamentos tenham a melhor câmara.
Se durante anos os Samsung lideravam nesta componente, a verdade é que marcas como a Huawei têm dado luta, tendo já conquistado o topo desta lista com as três câmaras do Huawei P20 Pro.

Com a concorrência a evoluir com os chips integrados para inteligência artifical, como é o caso dos NPU nos Kirin da Huawei, espera-se que a Google evolua o seu processador Pixel Visual Core de modo a que, mais uma vez, consiga extrair a melhor qualidade de imagem com a otimização do software para o sensor.
Além disso, a Google tem estado a trabalhar nos últimos anos no RAISR, uma tecnologia capaz de aumentar a nitidez das imagens e eliminar o ruído usando o machine learning. Aos poucos a tecnologia tem sido exibida, devendo estar pronta em breve para ser implementada assim que a equipa de investigadores consiga associar um processador com potência suficiente, o que poderá acontecer agora.

No entanto, enquanto as várias marcas se focam em aumentar o número de sensores, a Google tem feito uma verdadeira aposta no software que processa as fotografias. Apresentado no Pixel 2 XL, o Pixel Visual Core é um processador desenhado pela Google e pela Intel, que está unicamente dedicado à fotografia. Com uma integração de hardware e software, a marca americana conseguiu otimizar o processador unicamente para o sensor usado no Pixel 2 XL, tirando o máximo das suas capacidades.
Com o Pixel 3 XL a chegar, embora já praticamente todas as características sejam conhecidas, existe um pormenor que nunca foi falado, este mesmo processador.

Embora este equipamento apenas disponha de uma câmara, a Google prepara-se outra vez para tirar o máximo partido dela, muito com a ajuda de toda a tecnologia de hardware e software implementada.
Recentemente foram publicadas algumas supostas fotografias do Pixel 3 XL que não deixaram ninguém impressionado. No entanto, tal como aconteceu no último ano, a Google poderá ter o Pixel Visual Core desligado, sendo apenas ligado mais tarde, o que aconteceu com a primeira versão deste processador.
